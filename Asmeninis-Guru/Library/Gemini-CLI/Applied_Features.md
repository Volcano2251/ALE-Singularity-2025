# Naudojamos Gemini CLI Funkcijos

Šiame faile sekame, kokios dokumentacijos galimybės yra pritaikytos šiam projektui.

| Funkcija | Būsena | Aprašymas |
| :--- | :--- | :--- |
| **Hierarchinis Kontekstas** | ✅ Aktyvuota | Naudojamas `GEMINI.md` projekto taisyklėms. |
| **YOLO Režimas** | ✅ Aktyvuota | Naudojama `--yolo` vėliavėlė greitam darbui. |
| **Custom Ignore** | ✅ Aktyvuota | Sukurtas `.geminiignore` triukšmo mažinimui. |
| **Settings File** | ✅ Aktyvuota | Sukurtas `.gemini/settings.json` lokaliems nustatymams. |
| **MCP Servers** | ⏳ Svarstoma | Galimybė pridėti specifinius MCP serverius. |
| **System Prompt Override**| ✅ Aktyvuota | `.env` faile nurodytas kelias į `.gemini/system.md`. |
