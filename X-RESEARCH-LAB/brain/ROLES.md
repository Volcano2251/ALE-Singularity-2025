# ALE: Multi-Agent Role Definition

## 1. Roles
- **ARCHITECT**: Valdo `PLAN_MODDE.md`, nustato strategines kryptis.
- **RESEARCHER**: Naudoja `web_research` įgūdį, pildo `KNOWLEDGE_BASE.md`.
- **CODER**: Vykdo implementacijas, kuria skriptus.
- **REVIEWER (Autorater)**: Tikrina visų vaidmenų darbą pagal `CORE_ALGORITHM.md`.

## 2. Communication Protocol
Visi vaidmenys privalo rašyti savo būseną į `brain/COMMUNICATION.json`.

## 3. Workflow
Researcher randa info -> Architect įvertina -> Coder implementuoja -> Reviewer patvirtina.
